# AI Status Portal

A single, unified dashboard to view the real-time operational status of the most popular AI APIs and platforms.

## Features

- **Live Status Checking**: Automatically fetches the current status of supported services using their official Atlassian Statuspage JSON APIs.
- **Quick Links**: Centralizes access to the status pages of other major providers that do not offer public CORS-enabled JSON endpoints.
- **Premium Design**: Features a dark-themed, glassmorphism-inspired aesthetic with dynamic CSS micro-animations.

## Supported Platforms

### Dynamically Fetched (Real-time)
- **OpenAI**
- **Claude**
- **Deepseek**

### Direct Status Links (Static)
- **Gemini** (Google AI Studio)
- **Grok** (xAI)
- **Mistral AI**
